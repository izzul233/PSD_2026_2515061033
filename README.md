# SISTEM PENILAIAN BAHASA INGGRIS

# DESKRIPSI SINGKAT

Sistem penilaian bahasa inggris merupakan sebuah aplikasi berbasis Command Line Interface (CLI) yang dikembangkan menggunakan bahasa pemrograman Python untuk mengotomatisasi proses manajemen nilai akademik siswa. Proyek ini berfungsi sebagai implementasi praktis dari konsep Array 1 Dimensi (List), di mana sistem mengelola dua kumpulan data paralel untuk menyinkronkan identitas siswa dengan perolehan nilai mereka secara akurat. Fokus utama dari pengembangan sistem ini adalah menciptakan alur kerja yang tangguh (robust) melalui penerapan logika validasi data yang berlapis. Dengan mengintegrasikan struktur perulangan while True dan mekanisme error handling menggunakan blok try-except, sistem mampu melakukan mitigasi terhadap kesalahan input pengguna, seperti entri data non-numerik atau nilai di luar rentang standar 0 hingga 100, sehingga menjamin bahwa data yang diproses ke tahap selanjutnya benar-benar valid dan konsisten.

Secara teknis, program ini mendemonstrasikan efisiensi pemrograman modular dengan memisahkan logika bisnis penentuan kualifikasi nilai ke dalam fungsi khusus. Fungsi tersebut secara otomatis mengklasifikasikan nilai numerik ke dalam kategori huruf mutu (A hingga E) berdasarkan parameter yang telah ditentukan, yang kemudian diintegrasikan kembali ke dalam tabel laporan utama. Di tahap akhir eksekusi, sistem melakukan komputasi statistik untuk menghasilkan metrik performa kelas, termasuk kalkulasi rata-rata kumulatif serta identifikasi nilai ekstrem (maksimum dan minimum). Keunggulan sistem ini terletak pada kemampuannya untuk melakukan mapping indeks secara dinamis, sehingga identitas pemilik nilai tertinggi maupun terendah dapat dipanggil kembali dengan presisi tinggi. Seluruh informasi disajikan menggunakan teknik string formatting yang rapi, mencerminkan standar profesional dalam penyajian laporan data digital di bidang teknologi informasi.

# OUTPUT PROGRAM

Saat program dijalankan, pengguna akan disambut dengan header sistem yang bersih. Program kemudian akan meminta input nilai secara bergantian untuk setiap nama siswa yang ada di dalam array. Proses ini bersifat interaktif, di mana setiap baris input akan menyebutkan nama siswa secara spesifik untuk meminimalisir kesalahan entri data.Setelah seluruh data nilai berhasil divalidasi dan dikumpulkan, program akan mencetak sebuah tabel laporan yang terorganisir. Tabel ini terdiri dari empat kolom utama: Nomor Urut, Nama Siswa, Nilai Numerik, dan Grade (Huruf Mutu). Penggunaan teknik string alignment memastikan bahwa setiap kolom sejajar secara vertikal, memberikan kesan laporan profesional meski dijalankan di lingkungan terminal.Pada kolom Grade, sistem secara otomatis menampilkan hasil konversi berdasarkan logika fungsi get_grade. Hal ini memungkinkan pengguna untuk langsung melihat kategori pencapaian setiap siswa tanpa harus menghitung secara manual.Di bagian akhir output, terdapat blok ringkasan statistik yang memberikan nilai tambah pada laporan ini. Ringkasan ini mencakup Nilai rata-rata seluruh kelas yang ditampilkan dengan presisi dua angka di belakang koma.Nilai Tertinggi & Terendah Sistem tidak hanya menampilkan angka maksimal dan minimal saja, tetapi juga melakukan pemetaan balik ke array nama untuk menampilkan siapa siswa yang meraih nilai tersebut. Hal ini memberikan informasi yang lebih personal dan relevan bagi pengguna.
